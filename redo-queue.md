# Redo Queue -- Designs to Revisit

> Add designs you struggled with. Use spaced repetition: revisit on days 1, 3, 7, 14.

## How to Use

1. After attempting a design, if you struggled or needed to look at the solution, add it here
2. Revisit at increasing intervals (the dashboard automates this with Leitner boxes)
3. Each revisit: try designing from scratch, then compare with solution
